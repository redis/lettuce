# Rules

This project's coding rules are maintained **tool-neutrally** in `.agents/rules/` so
every agent shares one source. This file is the Claude Code entry point into them.

Read [`.agents/rules/rules.md`](../../.agents/rules/rules.md) for the index, and the
relevant rule set before editing — e.g. [`.agents/rules/java-rules.md`](../../.agents/rules/java-rules.md)
when editing Java source (`src/**/*.java`).
